# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately so it can be fixed before public disclosure.

- **Preferred:** use GitHub's [private vulnerability reporting](../../security/advisories/new) to open a draft advisory.
- **Or email:** jordan@jordankrueger.com

Please do **not** open a public issue for security problems. You'll get an acknowledgement, and we'll coordinate a fix and disclosure timeline with you.
